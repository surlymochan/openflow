# OpenFlow Goal

## Goal

Keep OpenFlow credible as an open-source agent delivery runtime: repo-owned
workflow contracts, executable verification gates, durable project context, and
honest benchmark evidence.

## Scope

- Keep `xflow:goal`, `xflow:yolo`, and `xflow:corps` aligned as one public
  workflow family.
- Keep CLI commands, docs, tests, release gates, and benchmark reports
  reproducible from source checkout.
- Keep public claims scoped to evidence generated by commands in this repo.

## Non-Goals

- Do not publish local operator state, raw private adoption evidence, secrets,
  or generated runtime artifacts.
- Do not claim universal superiority over other tools outside the disclosed
  benchmark rubric.
